# FlaskSkeleton

## SETUP:
Create a python file named dbconfig.py in /static/py and insert a dictionary object containing MySQL credentials.

Example:

`mysql = {
    "host":"localhost",
    "port":2251,
    "user":"root",
    "passwd":"100429659",
    "db":"crypto"
}`

## NOTE: This project has been setup to run/debug using VSCODE.
